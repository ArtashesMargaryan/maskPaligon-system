# Instructions

## Install

```sh
git clone https://github.com/armathai/playrix-mini-game-v2
cd playrix-mini-game-v2
npm install
```

## Usage

### to start a dev server

```sh
npm start
```

### to build single html file in dist folder

```sh
npm run build
```
